# 💰 CASH THRONE GLOBAL 🌍

A **multiplayer browser-based trading and real estate simulation game** where you compete globally to become a billionaire! Perfect for learning about stocks, properties, trading, and wealth management.

## 🎮 Play Now!

### 👉 **[PLAY ONLINE HERE](https://cherkurinandan.github.io/cashthrone-global)** 👈

*(No installation needed — just click and play!)*

---

## 🎮 Game Features

- **📈 Stock Trading** — Buy and sell 12 different stocks with dynamic prices that change every 2 seconds
- **🏠 Real Estate** — Invest in properties and earn daily rental income
- **🏢 Companies** — Start your own business (Tech, Retail, Food, Pharma, Entertainment) and generate passive income
- **🤝 Multiplayer Trading** — Trade items with other players with a 5% transaction fee
- **💬 Social Features** — Add friends and chat with other players in real-time
- **🏆 Global Leaderboard** — Compete with players worldwide and claim the top spot
- **💱 Multi-Currency** — Play from 8 countries (USA, UK, India, Japan, Germany, Brazil, China, Canada)
- **💾 Save/Load** — Export your game progress and import it to restore

## 🚀 How to Play

1. **Create Account** — Choose username, password, and country
2. **Start Trading** — Receive ₹5,000 starting capital
3. **Earn Money** — Buy stocks, invest in properties, start companies
4. **Trade & Compete** — Trade with friends and climb the leaderboard
5. **Win** — Build the highest net worth!

## 💾 Data Storage

- **Browser Storage** — Data saved locally in your browser (default)
- **Cloud Sync** — Optional Firebase sync for data backup
- **Export/Import** — Download your game as JSON and restore anytime

## 📋 Game Mechanics

### Starting Capital
- Begin with **₹5,000** in your chosen currency

### Ways to Earn Income
1. **Stock Trading** — Buy and sell stocks that fluctuate in price every 2 seconds
2. **Properties** — Purchase properties and earn daily rental income
3. **Companies** — Start a business and earn monthly passive income
4. **Player Trading** — Exchange goods with other players

### Investment Options

**Companies:**
- 💻 Tech Startup — ₹5,000 investment, ₹800/month income
- 🛍️ Retail Store — ₹3,000 investment, ₹500/month income
- 🍔 Food Business — ₹2,000 investment, ₹400/month income
- 💊 Pharma Co — ₹8,000 investment, ₹1,500/month income
- 🎬 Entertainment — ₹4,000 investment, ₹600/month income

**Stocks (12 Available):**
- Dynamic prices updated every 2 seconds
- Buy low, sell high to maximize profits
- Track price history and trends

## 🔧 Local Development

### Clone & Setup
```bash
git clone https://github.com/cherkurinandan/cashthrone-global.git
cd cashthrone-global
```

### Run Locally
```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (if installed)
npx http-server

# Then visit: http://localhost:8000
```
